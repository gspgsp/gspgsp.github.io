在实际开发过程中，需要各种开发工具，现在主要使用jetbrains全家桶，下面以 webstorm 的破解过程进行讲解:

  • 安装java环境

  • 下载工具jetbra

网站:https://3.jetbra.in/  
下载好了解压到指定盘，里面有一些脚本文件，其中:  
install-current-user.vbs - 适用于当前 Windows 用户。  
install-all-user.vbs - 适用于所有 Windows 用户。  
install.sh - 适用于 Linux 和 Mac  // ./scripts/install.sh，linux下的脚本执行完了，需要重新启动电脑，我这里用的是ubuntu22
  • 接下来，您需要将jetbra文件夹下的配置添加到 webstorm64.vmoptions文件中
-javaagent:~/Downloads/jetbra/ja-netfilter.jar=jetbrains  
在 linux 上，webstorm64.vmoptions 文件位于 ~/Downloads/WebStorm-223.7571.168/bin下

  • 还是到这个网站:https://3.jetbra.in/ 下载各个工具对应版本的激活码

  • 为应用添加图标，方便启动(这里以pycharm为例)

1>进入appications目录，新建xxx.desktop文件
guoshipeng@tianyi510s:~$ cd /usr/share/applications/
guoshipeng@tianyi510s:/usr/share/applications$ sudo vim py-charm.desktop
2>在xxx.desktop文件写入下面内容
[Desktop Entry]
Name=PyCharm
Exec=sh /home/guoshipeng/Downloads/pycharm-professional-2022.2.5/pycharm-2022.2.5/bin/pycharm.sh
Terminal=false
Type=Application
Icon=/home/guoshipeng/Downloads/pycharm-professional-2022.2.5/pycharm-2022.2.5/bin/pycharm.svg
Comment=PyCharm
Categories=Application;

注意上面Exec和Icon属性后面要修改为自己的WebStorm的安装目录对应的路径。
在上面配置完成后，打开右下角的应用显示里便能看到webstorm的启动图标了。

  • 启动应用，然后输入刚才负复制出来的激活码就可以了，目前2022.2版本的可以激活到2025年，估计2023.2的可以激活到2026年吧

  • 这里放几个激活码吧，都是2022.2.x的
