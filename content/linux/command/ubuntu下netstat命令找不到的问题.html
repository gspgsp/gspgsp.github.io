ubuntu下netstat命令找不到的问题：

vagrant@homestead:~$ netstat -a
-bash: netstat: command not found

解决办法:
使用netstat需要安装net-tools包
vagrant@homestead:~$ sudo apt install net-tools
然后就可以用了
