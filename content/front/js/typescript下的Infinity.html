typescript下的Infinity:
在 TypeScript 中，Infinity 是 JavaScript 的一个全局属性，表示“无穷大”。它是 number 类型的一个特殊值。你可以直接使用它，不需要导入或声明。
示例:
let max: number = Infinity;
console.log(max); // 输出: Infinity

console.log(1 / 0); // 输出: Infinity
console.log(-1 / 0); // 输出: -Infinity

console.log(Infinity > 1000000); // true
console.log(Infinity === Infinity); // true


特点：
Infinity 是一个数字类型的值，表示正无穷大。
-Infinity 表示负无穷大。
typeof Infinity === 'number' 是 true。
它是不可变的全局值，等价于 Number.POSITIVE_INFINITY。
