е…ідәҺPVе’ҢPVCзҡ„дҪҝз”Ё

з®ҖеҚ•жқҘиҜҙпјҡ

PVпјҲPersistentVolumeпјү жҳҜйӣҶзҫӨзә§еҲ«зҡ„еӯҳеӮЁиө„жәҗпјҢз”ұз®ЎзҗҶе‘ҳйў„е…ҲеҲӣе»әгҖӮе®ғд»ЈиЎЁе®һйҷ…зҡ„еӯҳеӮЁпјҲжҜ”еҰӮ NFSгҖҒiSCSIгҖҒдә‘еӯҳеӮЁзӯүпјүпјҢзӢ¬з«ӢдәҺд»»дҪ• Pod зҡ„з”ҹе‘Ҫе‘ЁжңҹеӯҳеңЁгҖӮ
PVCпјҲPersistentVolumeClaimпјү жҳҜз”ЁжҲ·зә§еҲ«зҡ„еӯҳеӮЁиҜ·жұӮгҖӮPod йҖҡиҝҮ PVC жқҘеЈ°жҳҺе®ғйңҖиҰҒеӨҡе°‘еӯҳеӮЁз©әй—ҙпјҢз„¶еҗҺ K8s дјҡиҮӘеҠЁеҢ№й…ҚеҗҲйҖӮзҡ„ PV з»‘е®ҡеҲ°иҝҷдёӘ PVCгҖӮ
е·ҘдҪңжөҒзЁӢжҳҜпјҡеҲӣе»ә PV вҶ’ еҲӣе»ә PVCпјҲжҢҮе®ҡеӨ§е°ҸгҖҒи®ҝй—®жЁЎејҸзӯүпјүвҶ’ PVC з»‘е®ҡеҲ° PV вҶ’ Pod жҢӮиҪҪ PVCгҖӮ
иҝҷз§Қи®ҫи®Ўзҡ„еҘҪеӨ„жҳҜи§ЈиҖҰдәҶеӯҳеӮЁз®ЎзҗҶе’Ң Pod дҪҝз”ЁпјҢз®ЎзҗҶе‘ҳиҙҹиҙЈжҸҗдҫӣеӯҳеӮЁиө„жәҗпјҢејҖеҸ‘иҖ…еҸӘйңҖеЈ°жҳҺйңҖиҰҒд»Җд№Ҳж ·зҡ„еӯҳеӮЁгҖӮ

PV е’Ң PVC зҡ„е…іиҒ”жҳҜиҮӘеҠЁзҡ„еҢ№й…ҚиҝҮзЁӢпјҡ

еҪ“дҪ еҲӣе»ә PVC ж—¶пјҢK8s дјҡж №жҚ® PVC зҡ„йңҖжұӮеҺ»жүҫеҗҲйҖӮзҡ„ PV иҝӣиЎҢз»‘е®ҡгҖӮеҢ№й…Қзҡ„жқЎд»¶дё»иҰҒжңүпјҡ

еӯҳеӮЁеӨ§е°ҸпјҡPV зҡ„е®№йҮҸиҰҒ >= PVC иҜ·жұӮзҡ„еӨ§е°Ҹ
и®ҝй—®жЁЎејҸпјҡPV е’Ң PVC зҡ„ accessModes иҰҒе…је®№
StorageClassпјҡеҰӮжһңйғҪжҢҮе®ҡдәҶпјҢиҰҒзӣёеҗҢпјҲжІЎжҢҮе®ҡзҡ„иҜқдёҚйҷҗеҲ¶пјү

е…·дҪ“жөҒзЁӢ:

еҲӣе»ә PVC
  вҶ“
K8s scheduler жЈҖжҹҘйӣҶзҫӨдёӯжүҖжңүеҸҜз”Ёзҡ„ PV
  вҶ“
еҜ»жүҫж»Ўи¶іжқЎд»¶зҡ„ PVпјҲеӨ§е°ҸеӨҹгҖҒaccessMode е…је®№зӯүпјү
  вҶ“
з»‘е®ҡеҲ°з¬¬дёҖдёӘз¬ҰеҗҲжқЎд»¶зҡ„ PV
  вҶ“
PVC зҠ¶жҖҒеҸҳдёә Bound

K8s PV/PVC дҪҝз”ЁеңәжҷҜжҖ»з»“

дёҖгҖҒеҝ«йҖҹеҜ№з…§иЎЁ

з»ҙеәҰж— зҠ¶жҖҒеә”з”ЁжңүзҠ¶жҖҒеә”з”Ё
йғЁзҪІж–№ејҸDeploymentStatefulSet
Pod иә«д»Ҫж— еӣәе®ҡиә«д»Ҫеӣәе®ҡиә«д»ҪпјҲPod-0гҖҒPod-1...пјү
еӯҳеӮЁж–№жЎҲеӨ–йғЁжңҚеҠЎжҲ–ж— еӯҳеӮЁжҜҸдёӘ Pod зӢ¬з«Ӣ PVC
жү©зј©е®№иҮӘз”ұжү©зј©йңҖиҰҒиҖғиҷ‘ж•°жҚ®дёҖиҮҙжҖ§
еёёи§Ғеә”з”ЁWebгҖҒAPIгҖҒзҪ‘е…іж•°жҚ®еә“гҖҒж¶ҲжҒҜйҳҹеҲ—гҖҒзј“еӯҳйӣҶзҫӨ
PV/PVCйҖҡеёёдёҚйңҖиҰҒеҝ…йңҖпјҲStatefulSet + volumeClaimTemplatesпјү
жҺЁиҚҗе®һи·өз”Ёдә‘жүҳз®ЎжңҚеҠЎз”Ёдә‘жүҳз®Ў жҲ– StatefulSet иҮӘз®Ў

дәҢгҖҒе®һжҲҳе»әи®®

еҜ№дәҺеҪ“еүҚ Go зҪ‘е…іеә”з”ЁпјҲж— зҠ¶жҖҒпјү

# жңҖз»ҲжҺЁиҚҗй…ҚзҪ®
apiVersion: apps/v1
kind: Deployment
metadata:
  name: go-gateway
spec:
  replicas: 4
  strategy:
    type: RollingUpdate
    rollingUpdate:
      maxSurge: 1
      maxUnavailable: 1
  selector:
    matchLabels:
      app: go-gateway
  template:
    metadata:
      labels:
        app: go-gateway
    spec:
      containers:
        - name: app
          image: go-gateway:0.1
          imagePullPolicy: IfNotPresent
          ports:
            - containerPort: 8080
          env:
            # ж•°жҚ®еә“
            - name: DB_HOST
              valueFrom:
                secretKeyRef:
                  name: db-secret
                  key: host
            - name: DB_PASSWORD
              valueFrom:
                secretKeyRef:
                  name: db-secret
                  key: password
            # зј“еӯҳ
            - name: REDIS_URL
              value: redis://redis.default.svc.cluster.local:6379

дёҚйңҖиҰҒ PV/PVCпјҒ еӣ дёәпјҡ

  • зҪ‘е…іжҳҜж— зҠ¶жҖҒзҡ„
  • ж•°жҚ®еә“з”Ёдә‘жүҳз®Ў
  • зј“еӯҳз”ЁеӨ–йғЁ Redis
  • ж—Ҙеҝ—иҫ“еҮәеҲ° stdout

еҜ№дәҺж•°жҚ®еә“пјҲжңүзҠ¶жҖҒпјү

# иҮӘз®Ў MySQLпјҲй«ҳзә§з”Ёжі•пјү
apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: mysql
spec:
  serviceName: mysql
  replicas: 3
  # ... й…ҚзҪ®и§ҒдёҠйқў StatefulSet дҫӢеӯҗ
  volumeClaimTemplates:
    - metadata:
        name: data
      spec:
        accessModes: [ "ReadWriteOnce" ]
        resources:
          requests:
            storage: 50Gi

  • ж— зҠ¶жҖҒеә”з”ЁпјҡдёҚз”Ё PVпјҢзӣҙжҺҘз”Ё Deployment + еӨ–йғЁжңҚеҠЎ
  • жңүзҠ¶жҖҒеә”з”Ёпјҡз”Ё StatefulSet + PVCпјҲиҮӘз®ЎпјүжҲ–зӣҙжҺҘз”Ёдә‘жүҳз®ЎжңҚеҠЎ
  • з”ҹдә§зҺҜеўғжңҖдҪіе®һи·өпјҡе°ҪйҮҸз”Ёдә‘еҺӮе•Ҷжүҳз®ЎжңҚеҠЎпјҢеҮҸе°‘иҝҗз»ҙжҲҗжң¬
