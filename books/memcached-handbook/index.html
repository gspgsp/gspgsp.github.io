Memcached дҪҝз”ЁжүӢеҶҢ

з®Җд»Ӣ

Memcached жҳҜдёҖдёӘй«ҳжҖ§иғҪзҡ„еҲҶеёғејҸеҶ…еӯҳеҜ№иұЎзј“еӯҳзі»з»ҹ,з”ЁдәҺеҠ йҖҹеҠЁжҖҒ Web еә”з”ЁзЁӢеәҸ,еҮҸиҪ»ж•°жҚ®еә“иҙҹиҪҪгҖӮе®ғйҖҡиҝҮеңЁеҶ…еӯҳдёӯзј“еӯҳж•°жҚ®е’ҢеҜ№иұЎжқҘеҮҸе°‘ж•°жҚ®еә“и®ҝй—®ж¬Ўж•°гҖӮ

ж ёеҝғзү№жҖ§

  • еҹәдәҺеҶ…еӯҳзҡ„й”®еҖјеӯҳеӮЁзі»з»ҹ
  • з®ҖеҚ•зҡ„еҚҸи®®и®ҫи®Ў(ж–Үжң¬еҚҸи®®е’ҢдәҢиҝӣеҲ¶еҚҸи®®)
  • еҲҶеёғејҸжһ¶жһ„ж”ҜжҢҒ
  • LRU(жңҖиҝ‘жңҖе°‘дҪҝз”Ё)ж·ҳжұ°зӯ–з•Ҙ
  • ж”ҜжҢҒеӨҡзәҝзЁӢ
  • й«ҳжҖ§иғҪ,дҪҺе»¶иҝҹ

е®үиЈ…дёҺй…ҚзҪ®

sudo apt-get update

еҗҜеҠЁ Memcached

еҹәжң¬еҗҜеҠЁ

memcached -d -m 64 -p 11211 -u memcache -l 127.0.0.1

еҸӮж•°иҜҙжҳҺ

  • -d: д»Ҙе®ҲжҠӨиҝӣзЁӢж–№ејҸиҝҗиЎҢ
  • -m: еҲҶй…Қзҡ„еҶ…еӯҳеӨ§е°Ҹ(MB),й»ҳи®Ө 64MB
  • -p: зӣ‘еҗ¬з«ҜеҸЈ,й»ҳи®Ө 11211
  • -u: иҝҗиЎҢз”ЁжҲ·
  • -l: зӣ‘еҗ¬ең°еқҖ
  • -c: жңҖеӨ§е№¶еҸ‘иҝһжҺҘж•°,й»ҳи®Ө 1024
  • -t: зәҝзЁӢж•°,й»ҳи®Ө 4
  • -P: дҝқеӯҳ PID зҡ„ж–Үд»¶и·Ҝеҫ„

й…ҚзҪ®ж–Үд»¶ж–№ејҸ(systemd)

# зәҝзЁӢж•°
-t 8

й«ҳзә§зү№жҖ§

дәҢиҝӣеҲ¶еҚҸи®®

Memcached ж”ҜжҢҒжӣҙй«ҳж•Ҳзҡ„дәҢиҝӣеҲ¶еҚҸи®®,зӣёжҜ”ж–Үжң¬еҚҸи®®е…·жңүд»ҘдёӢдјҳеҠҝ:

  • жӣҙе°‘зҡ„и§ЈжһҗејҖй”Җ
  • жӣҙзҙ§еҮ‘зҡ„ж•°жҚ®ж јејҸ
  • ж”ҜжҢҒжӣҙеӨҡж•°жҚ®зұ»еһӢ
  • еҶ…зҪ® CAS ж”ҜжҢҒ

еӨ§еӨҡж•°е®ўжҲ·з«Ҝеә“й»ҳи®ӨдҪҝз”ЁдәҢиҝӣеҲ¶еҚҸи®®гҖӮ

SASL и®ӨиҜҒ

еҗҜз”Ё SASL и®ӨиҜҒеўһејәе®үе…ЁжҖ§:

memcached -S -vv

й…ҚзҪ®з”ЁжҲ·еҗҚе’ҢеҜҶз Ғ:

echo "user:password" > /etc/sasldb2
saslpasswd2 -a memcached -c username

TLS/SSL ж”ҜжҢҒ

Memcached 1.5.13+ ж”ҜжҢҒ TLS еҠ еҜҶ:

memcached -Z -o ssl_chain_cert=/path/to/cert.pem \
          -o ssl_key=/path/to/key.pem

жү©еұ•еӯҳеӮЁеј•ж“Һ

Memcached ж”ҜжҢҒеӨ–йғЁеӯҳеӮЁеј•ж“Һ,еҸҜд»Ҙе°ҶдёҚеёёз”Ёж•°жҚ®еӯҳеӮЁеҲ°зЈҒзӣҳ:

memcached -o ext_path=/path/to/storage:1G

зј–зЁӢиҜӯиЁҖе®ўжҲ·з«Ҝ

values = mc.get_multi(['key1', 'key2'])

е®ўжҲ·з«Ҝеә”дҪҝз”ЁиҝһжҺҘжұ д»ҘеҮҸе°‘иҝһжҺҘејҖй”Җ:

й”®и®ҫи®ЎжңҖдҪіе®һи·ө

  1. дҪҝз”Ёз®ҖзҹӯдҪҶжңүж„Ҹд№үзҡ„й”®еҗҚ
  2. дҪҝз”Ёе‘ҪеҗҚз©әй—ҙеүҚзјҖ(еҰӮ user:1234)
  3. йҒҝе…ҚдҪҝз”ЁиҝҮй•ҝзҡ„й”®(жңҖеӨ§ 250 еӯ—иҠӮ)
  4. й”®еҗҚдҪҝз”Ё ASCII еӯ—з¬Ұ,йҒҝе…Қзү№ж®Ҡеӯ—з¬Ұ

ж•°жҚ®еҺӢзј©

еҜ№дәҺеӨ§ж•°жҚ®,е®ўжҲ·з«ҜеҸҜд»ҘиҮӘеҠЁеҺӢзј©:

mc.set('large_data', large_object, compress_threshold=1024)

еҲҶеёғејҸзӯ–з•Ҙ

дёҖиҮҙжҖ§е“ҲеёҢ

дҪҝз”ЁдёҖиҮҙжҖ§е“ҲеёҢз®—жі•еҲҶеёғж•°жҚ®еҲ°еӨҡдёӘиҠӮзӮ№:

зӣ‘жҺ§дёҺз»ҙжҠӨ

зӣ‘жҺ§жҢҮж Ү

  • cmd_get: GET е‘Ҫд»ӨжҖ»ж•°
  • cmd_set: SET е‘Ҫд»ӨжҖ»ж•°

  • bytes: еҪ“еүҚеӯҳеӮЁеӯ—иҠӮж•°
  • curr_connections: еҪ“еүҚиҝһжҺҘж•°
  • total_connections: жҖ»иҝһжҺҘж•°

и®Ўз®—зј“еӯҳе‘ҪдёӯзҺҮ

еёёи§Ғй—®йўҳжҺ’жҹҘ

еҶ…еӯҳдёҚи¶і

  • жЈҖжҹҘ evictions жҢҮж Ү
  • еўһеҠ еҶ…еӯҳеҲҶй…Қ:-m еҸӮж•°
  • дјҳеҢ–ж•°жҚ®иҝҮжңҹзӯ–з•Ҙ

иҝһжҺҘж•°иҝҮеӨҡ

  • жЈҖжҹҘ curr_connections
  • еўһеҠ жңҖеӨ§иҝһжҺҘж•°:-c еҸӮж•°
  • жЈҖжҹҘе®ўжҲ·з«ҜиҝһжҺҘжұ й…ҚзҪ®

жҖ§иғҪдёӢйҷҚ

  • жЈҖжҹҘ CPU дҪҝз”ЁзҺҮ
  • еўһеҠ зәҝзЁӢж•°:-t еҸӮж•°
  • дјҳеҢ–зҪ‘з»ңй…ҚзҪ®

жңҖдҪіе®һи·ө

зј“еӯҳзӯ–з•Ҙ

Cache-Aside(ж—Ғи·Ҝзј“еӯҳ)

def get_user(user_id):
    # е…ҲжҹҘзј“еӯҳ
    user = mc.get(f'user:{user_id}')
    if user is None:
        # зј“еӯҳжңӘе‘Ҫдёӯ,жҹҘж•°жҚ®еә“
        user = db.query(f'SELECT * FROM users WHERE id={user_id}')
        # еҶҷе…Ҙзј“еӯҳ
        mc.set(f'user:{user_id}', user, 3600)
    return user

Write-Through(еҶҷз©ҝ)

def update_user(user_id, data):
    # е…Ҳжӣҙж–°ж•°жҚ®еә“
    db.update(f'UPDATE users SET ... WHERE id={user_id}')
    # еҗҢжӯҘжӣҙж–°зј“еӯҳ
    mc.set(f'user:{user_id}', data, 3600)

Write-Behind(еҶҷеӣһ)

def update_user(user_id, data):
    # е…Ҳжӣҙж–°зј“еӯҳ
    mc.set(f'user:{user_id}', data, 3600)
    # ејӮжӯҘжӣҙж–°ж•°жҚ®еә“
    queue.add_task('update_db', user_id, data)

def get_data_with_lock(key):
    data = mc.get(key)
    if data is None:
        lock_key = f'lock:{key}'
        if mc.add(lock_key, 1, 10):  # иҺ·еҸ–й”Ғ
            try:
                data = db.query(...)
                mc.set(key, data, 3600)
            finally:
                mc.delete(lock_key)  # йҮҠж”ҫй”Ғ
        else:
            time.sleep(0.1)
            return get_data_with_lock(key)
    return data
