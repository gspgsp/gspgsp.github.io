CSS3 使用手册

CSS3 简介

CSS3 是 CSS 的最新标准,它引入了许多新特性,包括圆角、阴影、渐变、动画等,使得网页设计更加丰富和动态。

基本语法

selector {
  property: value;
}

选择器

基础选择器

/* 元素选择器 */
p { color: blue; }

/* 类选择器 */
.classname { font-size: 16px; }

/* ID选择器 */
#idname { background: yellow; }

/* 通配符选择器 */
* { margin: 0; padding: 0; }

属性选择器

/* 选择具有特定属性的元素 */
[type="text"] { border: 1px solid #ccc; }

/* 属性值以特定字符串开头 */
[href^="https"] { color: green; }

/* 属性值以特定字符串结尾 */
[src$=".png"] { border: 2px solid red; }

/* 属性值包含特定字符串 */
[class*="btn"] { padding: 10px; }

伪类选择器

/* 鼠标悬停 */
a:hover { color: red; }

/* 第一个子元素 */
li:first-child { font-weight: bold; }

/* 最后一个子元素 */
li:last-child { margin-bottom: 0; }

/* 第n个子元素 */
li:nth-child(2n) { background: #f0f0f0; }

/* 非特定元素 */
p:not(.special) { color: gray; }

伪元素选择器

/* 在元素前插入内容 */
h1::before { content: "★ "; }

/* 在元素后插入内容 */
h1::after { content: " ★"; }

/* 首字母样式 */
p::first-letter { font-size: 2em; }

/* 首行样式 */
p::first-line { font-weight: bold; }

/* 选中文本 */
::selection { background: yellow; }

盒模型

Box-sizing

/* 标准盒模型 */
.box1 {
  box-sizing: content-box; /* 默认值 */
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  /* 总宽度 = 200 + 40 + 10 = 250px */
}

/* 边框盒模型 */
.box2 {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  /* 总宽度 = 200px */
}

内外边距

/* 外边距 */
.element {
  margin: 10px;                    /* 四边 */
  margin: 10px 20px;               /* 上下 左右 */
  margin: 10px 20px 30px;          /* 上 左右 下 */
  margin: 10px 20px 30px 40px;     /* 上 右 下 左 */
}

/* 内边距 */
.element {
  padding: 15px;
  padding: 10px 20px 30px 40px;
}

盒阴影

  /* 水平偏移 垂直偏移 模糊半径 扩展半径 颜色 */
  box-shadow: 5px 5px 10px 2px rgba(0,0,0,0.3);
  
  /* 内阴影 */
  box-shadow: inset 0 0 10px rgba(0,0,0,0.5);
  
  /* 多重阴影 */
  box-shadow: 
    3px 3px 5px rgba(0,0,0,0.2),
    -3px -3px 5px rgba(255,255,255,0.5);

渐变

.linear-gradient {
  background: linear-gradient(to right, red, blue);
  background: linear-gradient(45deg, red, yellow, green);
  background: linear-gradient(to bottom, red 0%, yellow 50%, green 100%);
}

/* 径向渐变 */

  background: radial-gradient(ellipse at center, red, yellow, green);

/* 重复渐变 */

  background: repeating-linear-gradient(45deg, red, red 10px, blue 10px, blue 20px);

2D/3D 转换

2D 转换

.transform-2d {
  /* 平移 */
  transform: translate(50px, 100px);
  transform: translateX(50px);
  transform: translateY(100px);
  
  /* 旋转 */
  transform: rotate(45deg);
  
  /* 缩放 */
  transform: scale(1.5);
  transform: scale(2, 0.5);  /* x轴2倍, y轴0.5倍 */
  
  /* 倾斜 */
  transform: skew(20deg, 10deg);
  
  /* 组合变换 */
  transform: translate(50px, 50px) rotate(45deg) scale(1.2);
}

3D 转换

.transform-3d {
  /* 3D 平移 */
  transform: translate3d(50px, 100px, 150px);
  transform: translateZ(100px);
  
  /* 3D 旋转 */
  transform: rotateX(45deg);
  transform: rotateY(45deg);
  transform: rotateZ(45deg);
  transform: rotate3d(1, 1, 0, 45deg);
  
  /* 3D 缩放 */
  transform: scale3d(1, 2, 0.5);
  
  /* 透视 */
  perspective: 1000px;
}

/* 保留3D空间 */
.container-3d {
  transform-style: preserve-3d;
  perspective: 1000px;
}

变换原点

.transform-origin {
  transform-origin: center center;     /* 默认 */
  transform-origin: top left;
  transform-origin: 50% 50%;
  transform-origin: 100px 200px;
}

过渡与动画

过渡 (Transition)

.transition {
  width: 100px;
  transition: width 2s ease-in-out 0.5s;
  /* 属性 持续时间 速度曲线 延迟时间 */
}

.transition:hover {
  width: 300px;
}

/* 多个属性过渡 */
.multi-transition {
  transition: 
    width 2s,
    height 1s,
    background-color 3s;
}

/* 所有属性过渡 */
.all-transition {
  transition: all 0.3s ease;
}

动画 (Animation)

/* 定义关键帧 */
@keyframes slidein {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

/* 百分比关键帧 */
@keyframes rainbow {
  0% { background-color: red; }
  25% { background-color: yellow; }
  50% { background-color: green; }
  75% { background-color: blue; }
  100% { background-color: red; }
}

/* 应用动画 */
.animated {
  animation-name: slidein;
  animation-duration: 2s;
  animation-timing-function: ease-in-out;
  animation-delay: 0.5s;
  animation-iteration-count: infinite;  /* 无限循环 */
  animation-direction: alternate;       /* 往返运动 */
  animation-fill-mode: forwards;        /* 保持最后状态 */
  animation-play-state: running;        /* 播放状态 */
}

/* 简写 */
.animated-short {
  animation: slidein 2s ease-in-out 0.5s infinite alternate forwards;
}

动画速度曲线

.timing-functions {
  animation-timing-function: linear;           /* 匀速 */
  animation-timing-function: ease;             /* 慢-快-慢 */
  animation-timing-function: ease-in;          /* 慢速开始 */
  animation-timing-function: ease-out;         /* 慢速结束 */
  animation-timing-function: ease-in-out;      /* 慢速开始和结束 */
  animation-timing-function: cubic-bezier(0.1, 0.7, 1.0, 0.1);  /* 自定义 */
  animation-timing-function: steps(4, end);    /* 分步 */
}

弹性盒子布局

基础设置

.flex-container {
  display: flex;              /* 或 inline-flex */
  
  /* 主轴方向 */
  flex-direction: row;        /* 默认,水平方向 */
  flex-direction: row-reverse;
  flex-direction: column;     /* 垂直方向 */
  flex-direction: column-reverse;
  
  /* 换行 */
  flex-wrap: nowrap;          /* 默认,不换行 */
  flex-wrap: wrap;            /* 换行 */
  flex-wrap: wrap-reverse;    /* 反向换行 */
  
  /* 简写 */
  flex-flow: row wrap;
}

主轴对齐

.flex-container {
  /* 主轴对齐方式 */
  justify-content: flex-start;      /* 起点对齐 */
  justify-content: flex-end;        /* 终点对齐 */
  justify-content: center;          /* 居中 */
  justify-content: space-between;   /* 两端对齐 */
  justify-content: space-around;    /* 均匀分布 */
  justify-content: space-evenly;    /* 完全均匀 */
}

交叉轴对齐

.flex-container {
  /* 交叉轴对齐 */
  align-items: stretch;       /* 默认,拉伸 */
  align-items: flex-start;    /* 起点对齐 */
  align-items: flex-end;      /* 终点对齐 */
  align-items: center;        /* 居中 */
  align-items: baseline;      /* 基线对齐 */
  
  /* 多行对齐 */
  align-content: flex-start;
  align-content: center;
  align-content: space-between;
}

子项属性

.flex-item {
  /* 排序 */
  order: 1;                   /* 默认为0,越小越靠前 */
  
  /* 放大比例 */
  flex-grow: 1;               /* 默认为0 */
  
  /* 缩小比例 */
  flex-shrink: 1;             /* 默认为1 */
  
  /* 基础大小 */
  flex-basis: 200px;          /* 默认为auto */
  
  /* 简写 */
  flex: 1 1 200px;            /* grow shrink basis */
  flex: 1;                    /* 等同于 1 1 0% */
  
  /* 单独对齐 */
  align-self: center;         /* 覆盖容器的align-items */
}

性能优化建议

  1. 使用 transform 和 opacity 实现动画,这两个属性不会触发重排
  2. 避免使用通配符选择器 *,会影响性能
  3. 减少嵌套层级,建议不超过3层
  4. 使用 will-change 提示浏览器优化即将变化的属性
  5. 合理使用硬件加速,通过 transform: translateZ(0) 开启
  6. 压缩 CSS 文件,减少加载时间
  7. 使用 CSS Sprites,减少 HTTP 请求
/* will-change 示例 */
.will-change-example {
  will-change: transform, opacity;
}

/* 硬件加速 */
.gpu-accelerated {
  transform: translateZ(0);
  backface-visibility: hidden;
}

总结

CSS3 为网页设计带来了强大的功能,包括灵活的布局系统、丰富的视觉效果和流畅的动画。掌握这些特性能够帮助你创建现代化、响应式的网页界面。建议在实际项目中多加练习,逐步掌握这些技术。
